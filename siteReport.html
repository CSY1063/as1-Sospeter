Site Report

This is my personal portfolio project for assignment one submission. I drew inspiration from two portfolio websites, to which I have attached links at the very end. I incorporated some of the button designs and colour choices used in those websites for a professional appearance. It took me about five weeks to complete the entire project as I encountered various bugs along the way. With the assistance of tools such as Stack Overflow, I was able to resolve bugs during the development process. In instances where I couldnâ€™t find answers online, I consulted with my module tutor. For the background image, I utilized an abstract image that I found from a trusted website that offered copyright protection. I designed my logo using Canva and saved it as a PNG file in my files folder.

The module material helped me understand key techniques in web development. For example, I learned about the use of grid in the overall layout of the page, uploading my code to GitHub through Git, designing layouts for mobile optimization, using media queries, and implementing transitions and animations to enhance the visual appeal of my website. I also learned how to structure my HTML code and style the website using simple to advanced CSS selectors. By utilizing keyboard shortcuts, boilerplates, pre-defined functions, and Visual Studio Code extensions, I significantly increased my development speed. I sourced my fonts from the Google Fonts website, which offers free, reliable, and aesthetically pleasing font families.
